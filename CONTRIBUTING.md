# Contribuindo

Crie uma branch curta, instale `.[dev]` e mantenha cada alteração focada. Antes de enviar uma contribuição, execute `pytest`, `ruff check .` e `ruff format --check .`. Não adicione provedores que contornem DRM, autenticação, paywalls ou termos de serviço. Novos recursos precisam de testes sem dependência de internet.
